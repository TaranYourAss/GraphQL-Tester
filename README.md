# GraphQL-Tester

## Requirements  
cloudscraper  
plotext (if plotting data - used by default)
```
pip install cloudscraper, plotext
```

## Usage
```
python3 directive_overload.py --url https://{WEBAPP}.com/gql/v2 --cookies session=abc123 --plot-data True
```
